# Оценка сердечно-сосудистого риска

## Описание задачи
Прогнозирование риска сердечно-сосудистых заболеваний на основе медицинских и анкетных данных пациентов. Бинарная классификация: 0 — здоров, 1 — болен.

## Датасет
**Heart Disease Dataset (UCI/Cleveland)** — 1025 записей, 14 признаков.
Реальные клинические данные пациентов.

### Признаки:
| Признак | Описание |
|---------|----------|
| age | Возраст |
| sex | Пол (1=муж, 0=жен) |
| cp | Тип боли в груди (0-3) |
| trestbps | Давление в покое (мм рт.ст.) |
| chol | Холестерин (мг/дл) |
| fbs | Сахар в крови > 120 мг/дл |
| restecg | ЭКГ в покое (0-2) |
| thalach | Максимальный пульс |
| exang | Стенокардия при нагрузке |
| oldpeak | Депрессия ST |
| slope | Наклон ST |
| ca | Количество крупных сосудов |
| thal | Талассемия (0-3) |
| **target** | **Целевая переменная (0=здоров, 1=болен)** |

## Методы

### EDA (разведочный анализ данных)
- Загрузка и первичный осмотр данных
- Проверка пропусков (отсутствуют)
- Описательная статистика
- Визуализация распределений признаков (гистограммы, boxplot)
- Матрица корреляций

### Предобработка
- Разделение на признаки (X) и целевую переменную (y)
- Train/Test split (80/20, random_state=42)
- Стандартизация признаков (StandardScaler)

### Модели
| Модель | Accuracy | F1-score |
|--------|----------|----------|
| Logistic Regression | 0.80 | 0.81 |
| **Random Forest** | **0.99** | **0.99** |

### Выводы
- Random Forest значительно превосходит логистическую регрессию
- Нелинейные связи между признаками играют важную роль
- Наиболее значимые признаки: thalach, oldpeak, ca, cp
